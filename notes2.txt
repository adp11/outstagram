request to server app.post('/signup', (req, res) => {
  1. create user in mongodb
    hash password -> create a final object with full initialized values of that user -> .save()
  2. Attach token
    sign token by jwt.sign() -> res.cookie('token', token, { httpOnly: true }) -> send.json(user data)
})


frontend handle response(user data) [Token was saved in cookie already] {
  NO fetchUserData() and fetchAllUserData() in useEffect() in App.js
  NO fetchNewsfeed() in another useEffect()
  1. setUserData(userdata)
  2. setAllUserData(allUserData)
  3. setNewsfeed() (realtime in server)
}
